# What's Next

Ideas to expand your portfolio.

---

## Quick Wins

### RSS Feed
```bash
npm install @astrojs/rss
```
Let people subscribe to your blog.

### Sitemap
```bash
npm install @astrojs/sitemap
```
Better SEO, helps Google find your posts.

### Reading Time
Show "5 min read" on posts. Simple remark plugin.

### Table of Contents
Auto-generate TOC from headings for long posts.

---

## Content Ideas

### Series/Collections
Group related posts: "Security 101", "3D Printing Basics"

### Tags Page
`/tags/python` - show all posts with that tag

### Search
Add client-side search with Pagefind or Fuse.js

### Comments
- Giscus (GitHub discussions)
- Utterances (GitHub issues)
- No backend needed

---

## Features

### Dark/Light Mode
You have terminal theme. Add toggle for light mode?

### Analytics
- Plausible (privacy-friendly)
- Umami (self-hosted)
- Simple: just Coolify logs

### Newsletter
- Buttondown
- ConvertKit
- Collect emails for updates

### Project Gallery
Filter projects by category. Add live screenshots.

---

## Performance

### Image Optimization
```bash
npm install @astrojs/image
```
Auto-compress, lazy load, WebP conversion.

### Prefetching
```bash
npm install @astrojs/prefetch
```
Preload links on hover. Feels instant.
